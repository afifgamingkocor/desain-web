PENGENALAN WEB SERVER

Tugas Pemrograman Berbasis Jaringan

Apa itu Web Server?

Web Server adalah perangkat lunak atau perangkat keras yang bertugas menerima permintaan (request) dari client melalui browser dan mengirimkan halaman web yang diminta menggunakan protokol HTTP atau HTTPS.

Web Server berperan sebagai penghubung antara pengguna dan website yang diakses.

Fungsi Web Server

  • Menyimpan file website seperti HTML, CSS, JavaScript, gambar, dan video.
  • Menerima permintaan dari client.
  • Mengirimkan halaman web kepada pengguna.
  • Mengelola komunikasi antara client dan server.
  • Menjaga ketersediaan layanan website.

Contoh Web Server

  1. Apache HTTP Server
  2. Nginx
  3. Microsoft IIS
  4. LiteSpeed

Cara Kerja Web Server

Ketika pengguna mengetik alamat website pada browser, browser akan mengirimkan request ke web server. Setelah menerima request tersebut, web server akan memprosesnya dan mengirimkan halaman web yang diminta kepada browser untuk ditampilkan kepada pengguna.

Keamanan Web Server

  • Menggunakan HTTPS/SSL.
  • Memperbarui sistem dan software secara berkala.
  • Menggunakan firewall.
  • Membatasi hak akses pengguna.
  • Melakukan backup data secara rutin.
  • Memantau aktivitas server melalui log.

Kesimpulan

Web Server merupakan komponen penting dalam sistem jaringan yang berfungsi untuk melayani permintaan pengguna dan menyediakan halaman website yang dapat diakses melalui browser.
