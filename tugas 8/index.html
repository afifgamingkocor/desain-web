Mengenal Web Server

Web Server adalah perangkat lunak yang menerima request dari browser kemudian mengirimkan halaman website kepada pengguna melalui protokol HTTP maupun HTTPS.

Apa itu Web Server?

Web Server merupakan aplikasi yang bertugas menerima request dari client kemudian memprosesnya dan mengirimkan response berupa halaman website kepada browser.

Fungsi Web Server

📁 Menyimpan Website

Menyimpan file HTML, CSS, JavaScript, gambar, video, dan berbagai aset website.

🌐 Menerima Request

Melayani permintaan dari browser pengguna.

⚡ Mengirim Response

Mengirim halaman web kembali kepada client.
